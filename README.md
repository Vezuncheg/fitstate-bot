# FitState Bot — Railway Deploy

## Быстрый деплой на Railway

### 1. Создай репозиторий на GitHub
Загрузи файлы `bot.py`, `requirements.txt`, `Procfile` в новый репозиторий.
Назови его, например, `fitstate-bot`.

### 2. Зайди на Railway
- Открой railway.app → New Project → Deploy from GitHub repo
- Выбери репозиторий `fitstate-bot`

### 3. Добавь переменную окружения
В Railway → твой проект → Variables → добавь:
```
BOT_TOKEN = токен_от_BotFather
```

### 4. Railway сам запустит бота
Смотри логи в разделе Deployments — должно появиться:
```
FitState bot started
```

## Настройки бота (что менять в bot.py)

| Что менять | Где искать |
|---|---|
| Ссылка на оплату | `PAYMENT_URL = "..."` в начале файла |
| Дата потока | `[дата потока]` в функции `send_offer` |
| Цена | `[ЦЕНА]` в функции `send_offer` |
| Ссылка на канал потока | добавить в функцию после оплаты |

## Команды бота
- `/start` — начало, принимает параметр архетипа с сайта
- `/menu` — главное меню, показывает сохранённый результат
